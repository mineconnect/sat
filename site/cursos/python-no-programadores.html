La IA cambió la forma
de aprender a programar.

Este curso te lleva de cero a escribir scripts útiles para tu trabajo en pocas semanas, usando a Claude o ChatGPT como copiloto.

Pensado para administrativos, analistas y mandos medios que quieren manipular datos, generar reportes o automatizar tareas sin depender de IT. No requiere conocimientos previos.

Qué vas a aprender

  1. Fundamentos de Python explicados sin tecnicismos.
  2. Lectura y procesamiento de archivos Excel y CSV.
  3. Conexión con APIs y scraping básico.
  4. Generación automática de reportes en PDF y Excel.
  5. Cómo usar IA para resolver errores y acelerar el aprendizaje.
